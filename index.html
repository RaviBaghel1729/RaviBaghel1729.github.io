AW - ELASTIC BEANSTALK (Blue-Green Deployment)

AWS Elastic Beanstalk Blue-Green Deployment is a deployment strategy where two identical environments are used: Blue (current live version) and Green (new version). The new application version is deployed to the Green environment, tested safely without affecting users, and once verified, traffic is switched from Blue to Green. This approach enables zero-downtime deployments, easy rollback, and reduced deployment risk.

S3 STATIC WEB HOSTING

AWS S3 Static Website Hosting allows you to host static websites (HTML, CSS, JavaScript, images) directly from an S3 bucket without using a server. By enabling static website hosting and setting proper permissions, S3 serves web content with high availability, scalability, and low cost. It is commonly used for hosting frontend applications like React build files.
